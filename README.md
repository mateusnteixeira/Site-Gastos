# Site-Gastos
 Desenvolvendo um site para controle de gastos mensais

 *|Objetivos para a versão Pre-Alpha|*

2 Páginas:
*Página principal*(1)
*Página meses*(2)

1-
|-Nome do site
|-Menu(lateral)
  -->Acessar meses
  -->Acessar Gastos totais(anual)

2-
|-Selecionar o mês
   -->Tabela com os gastos
      -->Gastos separadados em: 
         |Nome|Valor|Vencimento|Parcelas(opcional)|
